# Glow — приложение для ухода за кожей

Glow помогает выстраивать персонализированные рутины ухода, отслеживать состояние кожи, водный баланс, получать аналитику и push-уведомления. Работает как PWA — можно установить на телефон и пользоваться офлайн.

## Основные возможности

- 📋 Создание и редактирование утренних/вечерних рутин с выбором продуктов и активных компонентов
- ✅ Ежедневное выполнение шагов рутины, оценка состояния кожи и реакций (раздражение, акне, увлажнение и др.)
- 💧 Расчёт нормы воды на основе веса, активности и погоды; добавление выпитой воды с визуальным прогрессом
- 📊 Аналитика: графики выполнения рутин, динамика состояния кожи, средние оценки реакций, прогресс по воде
- 🔔 Push-уведомления о времени выполнения рутины (утро/вечер) через Web Push API
- 📦 PWA: установка на устройство, работа офлайн (кэширование статики и API-ответов)
- 🐳 Полная контейнеризация через Docker Compose

## Запуск с Docker

1. Склонируйте репозиторий:
```bash
git clone https://github.com/mashroom-ah/glow_project.git
cd glow_project
```
2. Создайте файлы окружения:
   - `backend/.env` (см. образец ниже)
   - `frontend/.env.production` (публичный VAPID-ключ)

3. Запустите все сервисы:
```bash
docker compose up --build
```

4. Откройте в браузере http://localhost:3000

### Пример backend/.env

```bash
DB_NAME=glow_backend_db
DB_USER=postgres
DB_PASSWORD=postgres
DB_HOST=postgres
DB_PORT=5432
JWT_ACCESS_SECRET=your-secret-access-key
JWT_REFRESH_SECRET=your-secret-refresh-key
WEATHER_API_KEY=your_weather_api_key
VAPID_PUBLIC_KEY=BA...
VAPID_PRIVATE_KEY=...
```
### Пример frontend/env.production

```bash
VITE_API_URL=/api
VITE_VAPID_PUBLIC_KEY=BA... (тот же, что в бэкенде)
```

### Первоначальная настройка данных

При первом запуске автоматически применяются миграции и сидеры, создающие демо-пользователя (demo@example.com / Demo123!), группы продуктов, активные компоненты, шаблонные продукты, утренние и вечерние рутины для демо-пользователя, 30 дней логов с реакциями и оценками кожи.
